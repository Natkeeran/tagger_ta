ABOUT

	    The AutomaticTagger  for Tamil, tags the Tamil document as nouns, verbs, adjectives, adverb and postpositions. It uses the "Heuristic rules" and "Morphological rules". The input file should be TAB - encoded.

REQUIREMENTS

*J2RE 1.3   
* Operating System: Windows 95/98/ME/NT4/W2000/Linux

HOW TO RUN

Double click the jar file Kurippan.jar
OR
type "java -jar Kurippan.jar" at command prompt
